## Upload to GitHub

Upload these items to the root of `Praycres/Daily-Table`:

- `index.html`
- `style.css`
- `script.js`
- the entire `assets` folder

Commit the changes, wait 1–3 minutes, then hard-refresh:

https://praycres.github.io/Daily-Table/

## Dynamic panel rule — Option C

The bottom-right area uses one location:

1. Sunday Lunch Prep appears first on Sundays when prep fields exist.
2. Kitchen Whiteboard appears on weekends, or when Gentle Reminder is blank.
3. Otherwise Gentle Reminder appears.
